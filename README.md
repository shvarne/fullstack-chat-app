# Real-Time Chat Application

A full-stack real-time chat application built using the MERN stack and Socket.IO. Users can create accounts, log in securely, view online users, exchange messages instantly, and share images.

## Features

* User Authentication (JWT)
* User Signup and Login
* Secure Protected Routes
* Real-Time Messaging using Socket.IO
* Online/Offline User Status
* Image Sharing with Cloudinary
* Responsive User Interface
* State Management using Zustand

## Tech Stack

### Frontend

* React.js
* Zustand
* Axios
* Tailwind CSS
* DaisyUI
* Socket.IO Client

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Socket.IO
* Cloudinary

## Installation

Clone the repository:

git clone <repository-url>

Install dependencies:

Frontend

npm install

Backend

npm install

Create a .env file in the backend and add the required environment variables.

Run backend:

npm run dev

Run frontend:

npm run dev
